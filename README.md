📚 Stack Implementation in Python (Menu-Driven)

This project demonstrates a basic implementation of a Stack (LIFO - Last In First Out) using Python.
It includes a menu-driven program that allows users to perform stack operations interactively.

🚀 Features
Push element into stack
Pop element from stack
Peek top element
Display stack contents
Check if stack is empty
User-friendly menu-driven interface
🧠 Stack Concept

A Stack is a linear data structure that follows:

LIFO (Last In First Out)
